How to use
  • Local file: pick a .glb to preview it instantly — nothing gets committed.
  • From the repo: drop the file in assets/, type assets/name.glb, hit load.
  • Drag to orbit · scroll/pinch to zoom · it auto-spins.
Make models that behave on the web
  • Export a single .glb (textures embedded), not .gltf+.bin.
  • Keep it low-poly and textures small — aim for a few hundred KB, not tens of MB.
  • Apply transforms / real-world scale before export (or it imports huge/tiny).
  • Bake lighting/AO into the texture if you want it to read well unlit.
